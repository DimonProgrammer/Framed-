## ВОРОНКА A: КОНВЕРТЕР (лонгрид)

### Схема

```
Касание (Neurounit / Яндекс / ручные)
  → хук + предложение прочитать разбор
              ↓
Лонгрид (12 вопросов, ~10-15 мин чтения)
  Новизна → Выгоды → Кейсы → Корень проблемы →
  Снятие вины → Срочность → Экспертиза →
  Механизм → Оффер → Возражения → 3 варианта
              ↓
CTA: запись на стратсессию (форма / анкета)
              ↓
Стратсессия → Продажа
```

### Суть
Человек получает ссылку на продающий лонгрид, написанный по фреймворку "12 вопросов конвертера". Лонгрид прогревает, доказывает экспертизу, отрабатывает возражения. В конце — CTA на стратсессию.

### Для кого лучше всего
- Solution Aware (знают, что нужна система/помощь, выбирают какую)
- Скептики, которым нужно много доказательств перед звонком
- Те, кто предпочитает изучить материал перед принятием решения

### Каналы
- **Neurounit:** Автосообщение → ответ "да" → ссылка на лонгрид
- **Яндекс:** Поисковые запросы информационного типа ("как увеличить продажи в бутике") → лонгрид как лендинг
- **Ручные:** Как второе касание, если на прямое предложение не ответили

### Метрики

| Этап | Целевая конверсия |
|------|-------------------|
| Касание → открыли лонгрид | 20-30% |
| Открыли → дочитали до CTA | 40-50% |
| Дочитали → записались | 20-30% |
| Записались → пришли | 75-85% |
| **Касание → стратсессия** | **~5-7%** |

### Что нужно подготовить
- [ ] Лонгрид по фреймворку 12 вопросов (2-3 дня)
- [ ] Лендинг/форма записи на стратсессию
- [ ] Площадка для лонгрида (Telegraph / сайт / Tilda)

## ВОРОНКА B: ПРЯМАЯ (сразу на стратсессию)

### Схема

```
Касание (Neurounit / Яндекс / ручные)
  → прямое предложение стратсессии
              ↓
Лендинг стратсессии + квалификационная анкета
              ↓
Стратсессия → Продажа
```

### Суть
Самый короткий путь. Без промежуточного контента. Прямое предложение бесплатной стратсессии с экспертом.

### Для кого лучше всего
- Problem Aware (знают что проблема есть, готовы обсуждать)
- Решительные, деловые — "не хочу читать, хочу поговорить"
- Те, кто уже пробовал другие решения и ищет конкретную помощь

### Каналы
- **Neurounit:** Автосообщение с прямым оффером стратсессии
- **Яндекс:** Транзакционные запросы ("консультация fashion-ритейл", "обучение для владельцев бутиков") → лендинг стратсессии
- **Ручные:** Когда видим, что человек явно в проблеме (жалуется в соцсетях, снижает цены, etc.)

### Метрики

| Этап | Целевая конверсия |
|------|-------------------|
| Касание → заинтересовались | 8-12% |
| Заинтересовались → заполнили анкету | 50-70% |
| Заполнили → пришли | 70-80% |
| **Касание → стратсессия** | **~4-6%** |

### Что нужно подготовить
- [ ] Лендинг стратсессии с формой записи
- [ ] Квалификационная анкета (5-7 вопросов)
- [ ] Автоматическое подтверждение записи (email/Telegram)

## ВОРОНКА C: АУДИТ (персонализированный разбор)

### Схема

```
Предварительный анализ бизнеса (5-7 мин)
              ↓
Касание: "Посмотрел ваш бутик, нашёл X проблем"
  → 3 конкретных наблюдения по ИХ бизнесу
              ↓
Диалог → вовлечение
              ↓
Предложение стратсессии для полного разбора
              ↓
Стратсессия → Продажа
```

### Суть
Перед отправкой сообщения изучаем бизнес человека: Instagram, сайт, ценовую политику. Находим 3 конкретные проблемы. Отправляем сообщение с наблюдениями. Человек видит, что мы реально разобрались — доверие максимальное.

### Что анализируем перед отправкой

| Что смотрим | Что ищем |
|-------------|----------|
| Instagram бутика | Частота постов, качество, вовлечённость, наличие stories/reels |
| Ценовая политика | Частые скидки? Распродажи? Демпинг? |
| Клиентская работа | Есть ли программа лояльности? CRM-признаки? Персонализация? |
| Позиционирование | Понятно ли чем отличаются? Есть ли USP? |
| Команда | Видно ли сотрудников? Или "я одна"? |

### Шаблоны наблюдений (подставляем релевантные)

| Что видим | Формулировка |
|-----------|-------------|
| Частые скидки | Скидки при марже 50% = минус 40-60% прибыли с каждого чека. Есть альтернативы |
| Нет работы с базой | Нет видимой системы возврата клиентов — а 20% базы обычно дают 80% выручки |
| Слабый контент | Контент показывает товар, но не экспертизу. В премиуме покупают у людей |
| Нет сайта | Весь бизнес в одном канале — высокий риск |
| "Я одна" | Всё держится на вас — потолок роста и путь к выгоранию |

### Для кого лучше всего
- Problem Unaware → Problem Aware (даже те, кто не осознаёт проблему)
- Скептики ("а вы вообще понимаете мой бизнес?")
- Премиум-бутики с очевидными точками роста

### Каналы
- **Ручные:** Единственный канал. Это не масштабируется — 20-30 контактов/день макс.
- **НЕ подходит для Neurounit и Яндекс** — требует индивидуальной подготовки

### Метрики

| Этап | Целевая конверсия |
|------|-------------------|
| Отправили → ответили | 15-25% |
| Ответили → вовлеклись в диалог | 60-70% |
| Вовлеклись → записались на стратсессию | 40-50% |
| Записались → пришли | 80-90% |
| **Отправили → стратсессия** | **~8-12%** |

### Что нужно подготовить
- [ ] Чек-лист аудита (что смотреть в каждом бутике)
- [ ] Банк формулировок (15-20 типовых наблюдений)
- [ ] Шаблон с переменными для быстрой сборки сообщения

## КАСКАД: КОМБИНАЦИЯ ВОРОНОК НА ОДНОМ КОНТАКТЕ

Когда один подход не сработал — пробуем следующий:

```
ДЕНЬ 0:  Воронка C (Аудит) — для ручных рассылок
         Воронка B (Прямая) — для Neurounit
         "Посмотрел ваш бутик / Бесплатная стратсессия"

              ↓ нет ответа

ДЕНЬ 3:  Воронка A (Конвертер)
         "Есть разбор — как бутики растут в кризис. Скинуть?"

              ↓ нет ответа

ДЕНЬ 7:  Лид-магнит (калькулятор/чек-лист)
         "Бесплатный инструмент — калькулятор скрытой прибыли"

              ↓ нет ответа

ДЕНЬ 10: Финальное касание
         "Последний раз. Если не актуально — не буду писать"
```

**4 касания, 4 разных триггера.** Персонализация → экспертный контент → бесплатный инструмент → прямое закрытие.
