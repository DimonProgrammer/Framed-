## 1. ДИАГНОЗ: ЧТО НЕ ТАК С ШАБЛОНАМИ v1

### 1.1 Неправильный продукт
Шаблоны v1 продают **бесплатную стратсессию** и **чек-лист/калькулятор**. Актуальный продукт — **экспертный разбор за 1 500₽**. Промежуточный шаг — конвертер-лонгрид.

### 1.2 Неактуальный messaging
Везде «скрытые точки роста», «10-30% прибыли», «рост в кризис» — всё забраковано в реестре смыслов. Ни одной формулировки из согласованного списка.

### 1.3 Неподтверждённые цифры
«+22% средний чек», «+10% продажи», «затраты -200к» — используются как факт, хотя все ждут подтверждения от Екатерины. Правило: формулируем через действие, не через метрику.

### 1.4 Позиционирование от маркетолога
«Я работаю с владельцами бутиков — помогаю находить...» — маркетолог FRAMED = ноль доверия для owner с оборотом 20М+. Сообщение должно идти от лица или через контекст Екатерины.

### 1.5 Инфобиз-паттерны
- «Бесплатный чек-лист», «калькулятор скрытой прибыли» — классические лид-магниты, premium видела 100 раз
- «Актуально?», «Отправить?», «Интересно?» — одинаковые CTA в каждом шаблоне
- «Без продаж и звонков, просто инсайты» — красный флаг, именно так и продают

### 1.6 Нет warm-up стратегии
Все шаблоны — чистый cold. Для premium: warm-first (прогрев перед DM) даёт 22% принятия vs 5% у чистого cold.

### 1.7 Нарушения анти-ИИ
Шаблонные фразы, рубленые списки в email E5, однотипная структура во всех сообщениях.

## 2. ИССЛЕДОВАНИЕ: COLD OUTREACH В РФ-КОНТЕКСТЕ

### 2.1 Каналы

**Telegram — основной B2B-канал в РФ к 2026:**
- Прямой доступ к ЛПР без промежуточных слоёв
- Open rate значительно выше email (CTR до 18% для сегментированных рассылок)
- Нишевые fashion-сообщества реально работают
- Лимит: 4 096 символов на сообщение
- Ограничения: 20-50 сообщений/час, 500/сутки, интервал 30-90 секунд

**Instagram — дополнительный, для warm-up:**
- Fashion-аудитория там через VPN
- Лайки, комменты, реакции на сторис → потом переход в Telegram DM
- Не основной канал для холодного аутрича

**Email — для контактов с сайтов/LinkedIn:**
- Работает, но open rate ниже, чем в Telegram
- Подходит для более формальных касаний

**LinkedIn — слабый для fashion-ритейла в РФ:**
- Низкое adoption среди владельцев бутиков
- Можно убрать из приоритетов

### 2.2 Подход: Warm-First + Diagnose

**Шаг 1 — Прогрев (2-3 недели до DM):**
- Подписка на канал/аккаунт потенциального клиента
- Содержательные комментарии к постам (не «отличный пост!»)
- Реакции на сторис и публикации
- Результат: к моменту DM вы уже не незнакомец

**Шаг 2 — Первое касание (40-100 слов):**
- Назвать конкретную проблему, которую человек чувствует, но не сформулировал
- Показать, что знаешь его бизнес (название, сегмент, город)
- Никакого питча, никаких ссылок в первом сообщении — только вопрос
- Один чёткий CTA

**Результаты warm-first:** 22% принятия контакта, 7.22% ответов — значительно выше чистого cold (1-5%).

### 2.3 Данные по персонализации

- Персонализированные DM: +25% ответов vs шаблонные (HubSpot, 2024)
- AI-адаптация tone of voice под адресата: +32% вероятность ответа (Salesforce)
- Tailored контент: +40% склонность к покупке
- 76% топ-продавцов исследуют клиента ДО контакта
- 42% менеджеров работают без подготовки = конверсия ниже 2%

### 2.4 Оптимальная структура сообщений

**Первое касание (~50-80 слов для DM, ~100 для email):**
```
[Персонализированный крючок — первая строка видна в превью уведомления]
[1 предложение — конкретная проблема/наблюдение]
[1 предложение — что можем дать / контекст эксперта]
[1 вопрос — простой CTA]
```

**Follow-up цепочка (4 касания — x3 больше ответов vs одно сообщение):**

| Касание | День | Содержание |
|---------|------|-----------|
| #1 | 0 | Персонализированное первое сообщение |
| #2 | 2-3 | Ценный контент: кейс, инсайт, статистика |
| #3 | 5-7 | Другой angle/trigger |
| #4 | 10 | Уважительное закрытие |

**Тайминг:** вторник-четверг, 10-11 утра или 14-16, интервал 2-4 дня.

**Критическое правило:** Каждый follow-up — НОВАЯ ценность. Никаких «проверяю, получили ли вы моё сообщение». Один follow-up добавляет 40-50% дополнительных ответов.

### 2.5 Что заставляет fashion-ритейлеров отвечать

**Работает:**
1. Знание конкретных болей рынка — не «кризис в ритейле», а конкретика
2. Низкорисковое первое действие — 1 500₽ за диагностику vs сразу 170к
3. Позиция практика — Екатерина владеет сетью бутиков прямо сейчас
4. Антикризисный фрейм — не «обучение», а «как пройти кризис сильнее»
5. Социальное доказательство — конкретные кейсы (без неподтверждённых цифр)

**Убивает конверсию:**
1. Прямая реклама = спам
2. Покупные базы — «эффект крика»
3. Одно сообщение без follow-up — 48% продавцов бросают после первого
4. Слабая первая строка — если превью не зацепило, не откроют
5. Один канал — мультиканальный подход даёт x3 результат

### 2.6 Преодоление «антиинфобиз» скептицизма

Доверие к соцсетям в РФ: -20 п. (ВЦИОМ). 43% не доверяют, только 23% доверяют.

| Как НЕ говорить | Как говорить |
|----------------|-------------|
| «Курс для ритейлеров» | «Экспертный разбор с владельцем сети бутиков» |
| «Онлайн-обучение» | «Персональная диагностика бизнеса» |
| «Мы научим вас...» | «Разберём ваш бизнес, найдём где утекает» |
| «Бесплатный вебинар» | «1-на-1, 45 мин, конкретно по вашему бизнесу» |
| «Инфопродукт» | «Экспертное сопровождение от практика» |

### 2.7 Реалистичные бенчмарки

При 500 персонализированных сообщениях:

| Метрика | Cold | Warm-first |
|---------|------|------------|
| Ответы | 25-50 (5-10%) | 50-100 (10-20%) |
| Заинтересованных | 10-20 | 25-50 |
| Записей на разбор | 5-15 | 15-30 |
| Продаж практикума (20-30% конверсия) | 1-4 | 3-9 |

**Российские кейсы для сравнения:**
- Alter (B2B): 13.6% ответов через email-аутрич, 7.3% целевых лидов
- Completo (B2B-агентство): 7% конверсии в лид
- Telegram-рассылки (консультации): 1.5-2% конверсия при 1 000 контактов/день

### 2.8 AI-инструменты для персонализации

**Neurounit.ai:** End-to-end в Telegram — поиск, вовлечение, конвертация лидов.

**Coldy.ai:** AI-агент для российского рынка — находит компанию, проверяет соответствие ЦА, пишет персонализированное письмо. Цена лида: 70-200₽.

**Ключевой инсайт:** AI поднимает reply rate с 0.5-2% (шаблоны) до 6-20%+ (персонализация). Но 68% провальных AI-кампаний вызваны слабой предварительной аналитикой. Сначала качественная сегментация, потом AI-персонализация.

## 3. ПЛАН ШАБЛОНОВ v2

### 3.1 Ключевые изменения vs v1

| Было (v1) | Будет (v2) |
|-----------|-----------|
| Продукт: бесплатная стратсессия | Продукт: экспертный разбор 1 500₽ |
| CTA: чек-лист/калькулятор | CTA: лонгрид-конвертер или прямая запись на разбор |
| От имени: «я маркетолог FRAMED» | От имени: контекст Екатерины (действующий владелец) |
| Messaging: «точки роста», «рост 10-30%» | Messaging: «система vs интуиция», «дыры в бизнесе», «пройти кризис сильнее» |
| Цифры: +22%, +10%, -200к | Действия: «перестроила работу с базой», «нашла дыры в бюджете» |
| Каналы: email, LinkedIn, Telegram, звонки | Каналы: Telegram (основной), Instagram (warm-up + DM), email (доп.) |
| Подход: чистый cold | Подход: warm-first + cold |
| CTA: «Актуально?» / «Отправить?» везде | CTA: разные, под каждый шаблон |

### 3.2 Структура файла шаблонов

**Часть 1 — Warm-up протокол**
- Что делать ДО первого DM
- Как прогревать в Instagram / Telegram
- Таймлайн: 2-3 недели → DM

**Часть 2 — Telegram DM (основной канал)**
- T1: Наблюдение + конвертер (Funnel A)
- T2: Прямой оффер разбора (Funnel B)
- T3: Персональный аудит — 3 наблюдения (Funnel C)
- T4: Контринтуитивный инсайт (альт. hook)

**Часть 3 — Instagram DM (после warm-up)**
- I1: Из комментария в DM
- I2: Реакция на сторис → DM

**Часть 4 — Email (для контактов с сайтов)**
- E1: Conference insight + конвертер
- E2: Персональный аудит

**Часть 5 — Follow-up цепочка (единая для всех каналов)**
- FU1 (день 2-3): Новый инсайт / ценность
- FU2 (день 5-7): Другой angle
- FU3 (день 10): Уважительное закрытие

**Часть 6 — Каскадная стратегия**
- Обновлённая под новые воронки и продукт

### 3.3 Принципы текстов

1. **Первая строка = hook** — видна в уведомлении, решает откроют или нет
2. **Никакого «я помогаю»** — только контекст эксперта-практика Екатерины
3. **Ни одной неподтверждённой цифры** — действия, не метрики
4. **Все формулировки из реестра смыслов** — проверяются по messaging-registry.md
5. **Один CTA на сообщение** — каждый раз разный
6. **Анти-ИИ проверка** — каждый шаблон через Анти-ИИ-клише.md
7. **Максимум 80 слов для DM, 100 для email**
8. **Уважение к premium-аудитории** — без хайпа, без давления, деловой тон
