# Проект FRAMED — Инструкции для агента

## О проекте

**FRAMED** — образовательный проект для владельцев fashion-бизнесов в России. Не онлайн-школа в инфобиз-смысле — экспертное сопровождение от действующего практика.

### Ключевые лица

- **Екатерина Саввиди** — эксперт, 20+ лет в fashion-индустрии, владелец сети бутиков ReFrame в Москве, преподаватель НИУ ВШЭ. Лицо проекта и эксперт. Все тексты идут через её контекст.
- **Александр** — партнёр в FRAMED, отвечает за маркетинг и контроль процессов
- **Дима** (пользователь) — маркетолог и стратег, веб-дизайн, автоворонки, продажи

### Формат сотрудничества Димы

- 20% с каждой продажи (практикум 170к = 34к комиссия)
- Возможны варианты: фикс, фикс + процент, найм
- Также может вести продажи: проводить встречи, делать аудиты, продавать

## Правила текстов

### Позиционирование

- Все тексты идут **через контекст Екатерины** (действующий владелец бутиков)
- **Не** «я маркетолог FRAMED помогаю» — это ноль доверия для premium-аудитории
- Отправитель = человек из команды Екатерины, который связывает

### Цифры и кейсы

- **Ни одной неподтверждённой цифры** как факта — «+22% средний чек» и подобное НЕ используем, пока не подтвердит Екатерина
- Формулировать через действия: «перестроила работу с базой», «нашла дыры в бюджете»
- Кейсы: описывать что сделали, не привязывая конкретные метрики

### Анти-ИИ (обязательная проверка для каждого текста)

Полный чеклист → `01-context/Анти-ИИ-клише.md`

Ключевые паттерны, которые ЗАПРЕЩЕНЫ:
- Ложная глубина: «Проблема не в X. Проблема в Y»
- Рубленые фразы: 3+ предложений из 1-2 слов подряд
- Фальшивая близость: «Знакомо?», «Узнаёте себя?»
- Fake-энергия: «Спойлер:», «Plot twist:»
- Пустые связки: «Давайте разберёмся», «Важно понимать»
- Инфобиз-CTA: «Актуально?», «Отправить?», «Интересно?» (одинаковые)
- Зачёркнутые цены: ~~15 000₽~~ → 1 500₽
- Фейковый дефицит: «набирает группу», «осталось 3 места»
- Шаблонные лид-магниты: «бесплатный чек-лист», «калькулятор скрытой прибыли»

### Терминология

| Используем | НЕ используем |
|-----------|--------------|
| Блоки (диагностики) | Экраны |
| Экспертный разбор | Стратсессия, консультация |
| Практикум | Курс, онлайн-обучение |
| Действующий владелец | Коуч, наставник, эксперт (без контекста) |
| Пройти кризис сильнее | Рост на 10-30%, скрытые точки роста |
| Дыры в бизнесе | Зоны роста, точки роста |

## Воронки аутрича

### 3 параллельные воронки

| Воронка | Путь | Шаблоны |
|---------|------|---------|
| A — через контент | DM → конвертер-лонгрид → запись на разбор | T1, T4, I1, I2, E1 |
| B — прямой оффер | DM → диалог о разборе → запись | T2, E2 |
| C — персональный аудит | DM с 3 наблюдениями → диалог → разбор | T3 |

### Каналы (приоритет)

1. **Telegram** — основной B2B-канал в РФ (DM)
2. **Instagram** — warm-up (лайки, комменты, сторис) → переход в DM
3. **Email** — дополнительный, для контактов с сайтов
4. **LinkedIn** — убран из приоритетов (низкий adoption в fashion-РФ)

### Подход

- **Warm-first** (2-3 недели прогрева до DM) — x3-4 к ответам vs cold
- **Neurounit** — автоматизация для шаблонов T1, T2, T4
- **Ручной аутрич** — 20-30 контактов/день для T3

Шаблоны → `04-funnels/outreach-templates/first-touch-templates-v2.md`
Исследование → `04-funnels/outreach-templates/outreach-v2-plan.md`

## Стиль работы

- **Язык:** русский
- **Конкретность:** без общих слов, конкретные рекомендации
- **Фокус на ROI:** приоритет действиям для быстрых продаж
- **Исследования перед текстами:** сначала план и ресёрч, потом пишем
- **Итерации:** тексты проходят несколько раундов правок
- **Всё через реестр:** каждый текст проверяется по messaging-registry.md и Анти-ИИ-клише.md

## Запреты

- Не использовать неподтверждённые цифры как факт
- Не позиционировать от маркетолога — только через Екатерину
- Не использовать инфобиз-паттерны (см. анти-ИИ)
- Не делать выводов без данных
- Не игнорировать кризисный контекст
- Не привязывать тексты к одному сегменту (Wildberries, маркетплейсы)
