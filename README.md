# Проект FRAMED

> Онлайн-школа для fashion-ритейлеров | Стратегия выхода в прибыль

## Описание проекта

**Framed** — образовательная платформа для специалистов fashion-индустрии в России (владельцев бутиков, управляющих, байеров). Проект находится в стадии перезапуска с фокусом на антикризисные решения и реальные продажи.

**Период:** Январь-Март 2026
**Статус:** Разработка стратегии и воронок
**Цель:** Вывести курсы в прибыль через новую систему продаж

## Ключевые проблемы

- ✗ Низкая конверсия существующих воронок
- ✗ Продукты воспринимаются как "витаминка", а не решение проблем
- ✗ Кризис на рынке fashion-ритейла (падение спроса, рост налогов)
- ✗ Недостаточный фокус на практике и внедрении

## Наша стратегия

### 1. Позиционирование
Из "школы для ритейлеров" → в "антикризисного партнера с практическими решениями"

### 2. Продуктовая линейка
- **Мини-курсы** (low-ticket, первое касание)
- **Практикум** (core offer, ~170к, март 2026)
- **VIP-форматы** (личное ведение, внедрение, high-ticket)

### 3. Воронка продаж
```
ИИ-аутрич (Neurounit)
    ↓
Telegram-бот (прогрев)
    ↓
Лонгрид / Созвон (диагностика)
    ↓
Продажа
```

### 4. Фокус на quick wins
Приоритет — инструменты для быстрых продаж в течение 1-2 месяцев

## Структура проекта

```
framed/
├── 01-context/          # Контекст проекта, бриф, активы
├── 02-research/         # Исследования рынка, ЦА, конкурентов
├── 03-strategy/         # Стратегия, позиционирование, офферы

```
